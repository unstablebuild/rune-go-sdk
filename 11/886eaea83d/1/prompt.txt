# Sandbox Implementation Plan

## Context

Rune needs a sandbox implementation to securely execute commands from extensions. The sandbox must be **transparent** to callers - they call `executor.Start(ctx, cmd)` and the command runs in an isolated VM instead of on the host. This protects against malicious or buggy extension code that could compromise the user's system.

**Key decisions made:**
- Scope: Extensions only (via `workspaceapi.Executor` interface)
- Integration: Library in this SDK (not standalone service)
- Technology: Matchlock-based (Firecracker on Linux, Virtualization.framework on macOS)
- Lifecycle: One sandbox per workspace, created via constructor, destroyed on Close()
- Platforms: Linux x86_64, Linux arm64, macOS arm64 (macOS x86_64 low priority)

## Architecture Overview

```
Extension calls executor.Start(ctx, cmd)
           │
           ▼
┌─────────────────────────────────────┐
│     sandbox.Executor                │  Implements workspaceapi.Executor
│  (created via sandbox.New())        │
└──────────────┬──────────────────────┘
               │ vsock
               ▼
┌─────────────────────────────────────┐
│     Guest Agent (in VM)             │  Receives commands, streams I/O
│  - Executes commands                │
│  - Streams stdin/stdout/stderr      │
│  - Forwards signals                 │
└─────────────────────────────────────┘
```

## Key Types

```go
// workspaceapi.Executor interface we must implement
type Executor interface {
    Start(context.Context, Cmd) (Pid, error)
    Signal(Pid, syscall.Signal) error
    io.Closer
}

// sandbox/config.go
type Config struct {
    WorkspaceRoot string
    Policy        *policy.Policy
    Resources     *ResourceConfig
    Image         string
}

// sandbox/executor.go - Main implementation
type Executor struct {
    runtime Runtime
    vm      VM
    agent   *sandboxrpc.Client
    config  Config
    log     *slog.Logger
}

// New creates a sandbox executor. VM is started immediately.
func New(ctx context.Context, config Config) (*Executor, error) {
    runtime, err := NewRuntime()
    if err != nil {
        return nil, fmt.Errorf("create runtime: %w", err)
    }

    vm, err := runtime.Create(ctx, &config)
    if err != nil {
        return nil, fmt.Errorf("create vm: %w", err)
    }

    agent, err := sandboxrpc.NewClient(ctx, vm)
    if err != nil {
        vm.Close()
        return nil, fmt.Errorf("connect agent: %w", err)
    }

    return &Executor{
        runtime: runtime,
        vm:      vm,
        agent:   agent,
        config:  config,
    }, nil
}

func (e *Executor) Start(ctx context.Context, cmd workspaceapi.Cmd) (workspaceapi.Pid, error) {
    return e.agent.StartCommand(ctx, cmd)
}

func (e *Executor) Signal(pid workspaceapi.Pid, sig syscall.Signal) error {
    return e.agent.Signal(pid, sig)
}

func (e *Executor) Close() error {
    e.agent.Close()
    return e.vm.Close()
}
```
