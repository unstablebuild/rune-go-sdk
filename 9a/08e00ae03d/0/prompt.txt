## Subcommands

All commands support `-F` flag (json, table, Go template). Positions are **0-indexed** (matching LSP protocol).

### 1. `lsp hover <file> <line> <col>`
- **Calls:** `lsp.Hover(ctx, HoverParams{...})`
- **Returns:** `*Hover` (nullable) — contents markup + optional range
- **Default output:** Print contents value text
- **JSON:** `{"success":true, "result":{"contents":{...}, "range":{...}}}`

### 2. `lsp definition <file> <line> <col>`
- **Calls:** `lsp.Definition(ctx, DefinitionParams{...})`
- **Returns:** `[]Location`
- **Default output:** One per line: `<uri> <startLine>:<startChar>-<endLine>:<endChar>`
- **Table fields:** URI, StartLine, StartChar, EndLine, EndChar

### 3. `lsp references <file> <line> <col>`
- **Calls:** `lsp.References(ctx, ReferenceParams{..., Context: {IncludeDeclaration: c.includeDecl}})`
- **Extra flag:** `-d` — include declaration
- **Returns/output:** Same format as `definition`

### 4. `lsp symbols <file>`
- **Calls:** `lsp.DocumentSymbol(ctx, DocumentSymbolParams{...})`
- **Returns:** `[]DocumentSymbol` (hierarchical with Children)
- **Default output:** Indented tree with name, kind, range
- **JSON:** Full tree structure preserved
- **Table fields:** Name, Detail, Kind, Depth, StartLine, StartChar, EndLine, EndChar (flattened)

### 5. `lsp workspace-symbols <query>`
- **Calls:** `lsp.WorkspaceSymbol(ctx, WorkspaceSymbolParams{Query: query})`
- **Returns:** `[]SymbolInformation`
- **Default output:** `<name> <kind> <uri>:<line>:<char>`
- **Table fields:** Name, Kind, URI, StartLine, StartChar

### 6. `lsp diagnostics <file>`
- **Calls:** `lsp.Diagnostic(ctx, DocumentDiagnosticParams{...})`
- **Returns:** `DocumentDiagnosticReport`
- **Default output:** `[severity] line:col: message (source, code)`
- **Table fields:** Severity, StartLine, StartChar, EndLine, EndChar, Message, Source, Code

### 7. `lsp rename <file> <line> <col> <new-name>`
- **Calls:** `lsp.Rename(ctx, RenameParams{..., NewName: newName})`
- **Returns:** `*WorkspaceEdit`
- **Default output:** Summary per file: `<uri>: N edits`
- **JSON:** Full WorkspaceEdit with changes map

### 8. `lsp code-actions <file> <line> <col>`
- **Calls:** `lsp.CodeAction(ctx, CodeActionParams{..., Range: zeroWidthAtPos, Context: {Diagnostics: []}})`
- **Returns:** `[]CodeAction`
- **Default output:** `[kind] "title" (has edit/command)`
- **Table fields:** Title, Kind
